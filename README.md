# Laravel_Login

### Laravel 8

RUN:
```
1. composer update
2. php artisan key:generate
3. php artisan migrate
4. php artisan db:seed
5. npm install 
6. npm run dev
7. php artisan serve
```
Untuk username dan password login bisa dilihat di `/database/seeders/UserSeeder.php`;
